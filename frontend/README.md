## Build Setup

### install dependencies
``` bash
npm install
```
To check if depnendancies need an update
``` bash
ncu -u
npm install
``` 


### serve with hot reload at localhost:8080
``` bash
npm run start
```

### build for dev with sanic
``` bash
npm run dev
```

### build for production with minification
``` bash
npm run build
```

or

``` bash
npm run prod
```
